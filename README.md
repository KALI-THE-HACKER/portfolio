# Personal Portfolio — Source for Automatic Deployment

This repository contains the source code for my personal portfolio website and is used for automatic deployment.

Purpose

- This repo holds the static site that showcases projects, contact information, a résumé, and other personal work.
- It is the canonical source used to build and publish the portfolio site automatically.

Hosting & Deployment

- The site is hosted on Cloudflare Pages.
- The live site is served from a subdomain of my custom domain:

  portfolio.luckylinux.xyz

- Changes pushed to the repository (main branch) are automatically deployed by Cloudflare Pages.

Tech Stack

- Static HTML, CSS, and JavaScript (no server-side runtime required).
- Any build or bundling (if added later) is handled before deployment by Cloudflare Pages build settings.
